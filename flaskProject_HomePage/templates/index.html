지진알림 API

서비스지향 컴퓨팅 프로젝트 추명엽, 김호성

입력한 좌표의 법정동을 기준으로 지진 발생의 횟수를 알려줍니다.
진앙지의 거리와 지진의 규모를 변수로 법정동의 지진 발생 횟수를 계산합니다.

API 소개

지진 발생 횟수 DB

법정동 중심 좌표

대한민국 법정동 EPSG:4326 형식의 구역도에서 중심좌표를 구하였습니다.

진앙지와의 거리

법정동의 중심좌표와 진앙지의 거리를 'vincenty 알고리즘'로 구하였습니다.

지진의 영향

'oh의 지진 감쇠식'을 이용하여 진앙지의 지진이 법정동에 주는 영향을 계산하였습니다.

DB 구축

지진 발생 표현

지도 표현

'Folium'을 활용하여 지도를 표현합니다. 해당 지도에는 입력좌표의 법정동, 지진 횟수 정보가 Marker와 POPUP으로 표시됩니다.

이미지로 변환

'Folium'으로는 html 형식으로 저장되기 때문에, 'selium'의 캡쳐 기능을 이용하여 이미지 형식으로 변환하였습니다.

이미지의 크기 조절

지도에서 법정동의 중심좌표에 표현한 정보가 잘 보이도록 이미지를 조절하였습니다.

정보 표현

지도 이미지 전송

법정동을 중심으로 지진 정보가 표현된 지도 이미지를 네이버MMS API로 전송합니다.

지도 전송

입출력

입력

GET
3.34.144.159:5000/'phone'/'lat'/'lon'

'phone' : 보낼 전화번호 ex)01012345678
'lat' : 십진수 형태(Decimal Degree)의 대한민국 경도
'lon' : 십진수 형태(Decimal Degree)의 대한민국 위도

입력

출력

Jason
{
    "sendTo": "01012345678",
    "area": "\uce68\uc0b0\ub3d9",
    "sendMMS": "success/fail",
    "mags": [
                {
                "mag3" : 10,
                "mag4" : 5,
                "mag5" : 3,
                "mag6" : 2,
                "mag7" : 0,
                "mag8" : 0,
                "mag9" : 0,
                "mag10" : 0,
                "mag11" : 0,
                "mag12" : 0
                }
            ]
}
반환값 설명

"sendTo" : MMS를 받는 전화번호이고 string형입니다.
"area" : 법정동의 이름으로 반환값은 한글이고 UTF-8 형식입니다.
"sendMMS" : 해당 법정동의 지진 정보가 메시지로 전송되었는지 확인하는 것으로 반환값은 boolean형입니다.
"mags" : 해당 법정동의 지진 횟수가 저장된 배열입니다. 진도 3~12의 발생 횟수를 int형으로 반환합니다.

이미지 크기 : 426x380
